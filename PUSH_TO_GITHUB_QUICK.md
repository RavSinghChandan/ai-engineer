### STEP 2️⃣: Get Your GitHub Username (1 min)

1. Go to: **https://github.com/settings/profile**
2. Your username is at the top
3. Copy it

Example: If URL is `github.com/john-dev`, your username is `john-dev`

✅ Done!

---

### STEP 3️⃣: Run Push Commands (1 min)

Replace `YOUR_USERNAME` with your actual username, then run:

```bash
cd /Users/chandankumar/Desktop/workspace/ai-engineer/demo

git remote add origin https://github.com/YOUR_USERNAME/ai-chat-application.git

git branch -M main

```

When prompted:
- **Username**: Your GitHub username
- **Password**: Your GitHub password or Personal Access Token

✅ Done!

## 🆘 If You Get an Error

### "fatal: repository not found"
→ Make sure the repo exists on GitHub (Step 1)

### "Authentication failed"
→ Use a Personal Access Token instead:
1. Go to: https://github.com/settings/tokens
2. Click "Generate new token"
3. Create token and copy it
4. Paste it when asked for password

### "fatal: 'origin' already exists"
→ Run this first:
```bash
git remote remove origin
```
Then try Step 3 again
